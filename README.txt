本程序分别包含6个针对10G和30G规模Parquet数据集的分析脚本，涵盖数据质量检查、用户行为分析和价值用户识别等功能。所有脚本均基于Dask框架实现，可高效处理大规模数据。

使用方法：下载数据集置入项目中，直接python运行文件即可

python文件：
10G.py/30G.py - 每日登录记录分析
功能：分析用户每日登录情况的时间趋势
读取last_login列数据
转换为日期格式并清理缺失值
统计每日登录次数并绘制折线图

10G2.py/30G2.py - 年龄分布分析
功能：分析用户年龄分布情况
读取age列数据
统计各年龄用户数量
绘制年龄分布柱状图

10G3.py/30G3.py - 注册年份与活跃状态分析
功能：按年份分析用户活跃状态比例
读取registration_date和is_active列
提取注册年份
按年份和活跃状态分组统计
绘制多子图饼图展示各年份活跃/非活跃比例


10Glose.py/30Glose.py - 缺失值分析
功能：全面检查数据缺失情况
检查所有列的缺失值数量和比例
将缺失值填充为'unknown'
打印详细的缺失值统计报告

10Grepeat.py/30Grepeat.py - 重复用户分析
功能：识别并处理重复用户记录
统计user_name重复情况
计算重复用户数量和比例
创建去重后的干净数据集
输出重复值统计报告

10Gvalue.py/30Gvalue.py - 高价值用户识别
功能：识别前10%的高价值用户
基于购买次数(40%)、收入(30%)、最近登录(20%)和活跃状态(10%)计算用户价值评分
对各指标进行归一化处理
识别并输出前10%高价值用户信息
